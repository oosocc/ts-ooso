## 初始化项目

### 添加Antd
`
yarn add antd 
`
#### 删除 Antd
`
yarn remove antd
`
### 添加Mui
`yarn add @material-ui/core@next @emotion/react @emotion/styled`

### 添加 axios
` 
yanr add axios
`

### 添加路由
`
yarn add react-router-dom @types/react-router-dom
`
